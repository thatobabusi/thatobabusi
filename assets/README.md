# assets/

Committed images referenced by the profile [README](../README.md).

| File | What it is | Maintained how |
|:-----|:-----------|:---------------|
| `code-*.svg` | Tokyo Night code cards shown in the About Me section | **Generated — do not edit by hand.** Edit the snippets in [`scripts/generate_code_cards.py`](../scripts/generate_code_cards.py) and re-run it (the [Code Cards workflow](../.github/workflows/code-cards.yml) also regenerates them automatically when the generator changes) |
| `Thato-Read-me.svg` | Travel history map | Hand-maintained |
| `Thato-Read-Me-Travel-Wishlist-Word-Cloud.svg` | Travel wishlist word cloud | Hand-maintained |

Everything here is served straight from the repo with relative paths — no
third-party image service involved, so these can't break or rate-limit.

Theme reference (gradient, card colors) lives in [docs/README.md](../docs/README.md#theme).
